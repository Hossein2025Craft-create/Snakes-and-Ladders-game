# Our-game-on-the-stairs
بازی مار و پله آنلاین یک بازی گرافیکی جذاب ساخته‌شده با Python و Tkinter است که از حالت دو نفره آنلاین پشتیبانی می‌کند. بازیکنان با انداختن تاس روی صفحه حرکت می‌کنند، از نردبان‌ها بالا می‌روند و مراقب مارها هستند! این نسخه دارای رابط کاربری زیبا، ارتباط آنلاین با WebSocket، انیمیشن، چت و تجربه‌ای سرگرم‌کننده و رقابتی است. 🐍🪜🏆
